Tezos Chat Rooms

Where the Tezos community chats

Here are some popular1 places to chat about Tezos

Riot / Matrix

  • riot.tzchat.org is the Riot host for the new tzchat.org server in the Matrix federation, home to chat rooms with the bantering of Tezos OGs since early 2017. Register there for a tzchat.org account; login and launch Riot from there to access rooms in Matrix. Get started.

  • riot.im is the Riot host for existing matrix.org accounts, accessing the same Matrix rooms as riot.tzchat.org.

Telegram

  • Tezos Platform is the TG home to many Tezos OGs who like lots of the same gifs2 in their chat.
  • Tezos Announcements is a curated announcement-only room of Tezos news.

IRC

  • #tezos channel on freenode IRC (bridged to #tezos room on Matrix) was started by Tezos core devs back in the day and still has top tech minds lurking and helping out.
    /join #tezos

Slack

  • Obsidian runs a Baking Slack. Important announcements to the baking community often happen there first. Bakers jointly resolve tech problems, including Ledger issues. If you bake Tezos you want to at least follow announcements there. Request an invitation at tezos@obsidian.systems.

Reddit

The /r/tezos subreddit is active and has an associated subchat. Talk about prices in /r/tezostrader. Talk about baking services and delegation in /r/tezosdelegate.

Discord

The Tezos Blockchain Discord server is getting active. Here’s an invitation link.

Discourse

The Tezos Agora Forum is a place to discuss and learn about the long-term direction of the Tezos project.

Tezos Stack Exchange

While the above chat rooms can be a good source of interactive help, the Tezos StackExchange site has the answers to many questions about Tezos and is a good place to post new questions.

  1. No doubt what’s here is a biased view of what’s available. Additions and corrections to the list of chat places are welcome. ↩︎

  2. Just kidding. ↩︎
